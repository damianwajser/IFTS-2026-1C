# MiPrimeraApp

Proyecto de ejemplo del curso **Desarrollo de Aplicaciones para Dispositivos Móviles** — IFTS Nº16, 2026.

## Requisitos previos

- [Node.js](https://nodejs.org/) (versión 18 o superior)
- [Expo Go](https://expo.dev/go) instalado en tu celular (Android o iOS)

## Instalación

1. Cloná el repositorio:
   ```bash
   git clone https://github.com/damianwajser/IFTS-2026-1C.git
   cd IFTS-2026-1C/MiPrimeraApp
   ```

2. Instalá las dependencias:
   ```bash
   npm install
   ```

## Levantar la app

```bash
npm start
```

Esto abre el servidor de Expo. Escaneá el código QR con la app **Expo Go** desde tu celular (tiene que estar en la misma red Wi-Fi que la computadora).

## Otras opciones

```bash
npm run android   # Abre en emulador Android
npm run ios       # Abre en simulador iOS (solo Mac)
npm run web       # Abre en el navegador
```

Repositorio de proyectos del curso. Cada carpeta corresponde a una app trabajada en clase.

## Proyectos

| Carpeta | Clase | Contenido |
|---------|-------|-----------|
| [MiPrimeraApp](./MiPrimeraApp) | 1 y 2 | Componentes, props y estilos básicos |

## Requisitos previos

- [Node.js](https://nodejs.org/) v18 o superior
- [Expo Go](https://expo.dev/go) instalado en el celular

## Cómo levantar cualquier proyecto

1. Cloná el repositorio:
   ```bash
   git clone https://github.com/damianwajser/IFTS-2026-1C.git
   ```

2. Entrá a la carpeta del proyecto que querés correr:
   ```bash
   cd IFTS-2026-1C/MiPrimeraApp
   ```

3. Instalá las dependencias:
   ```bash
   npm install
   ```

4. Levantá el servidor:
   ```bash
   npm start
   ```

5. Escaneá el QR con **Expo Go** desde tu celular (misma red Wi-Fi que la computadora).
